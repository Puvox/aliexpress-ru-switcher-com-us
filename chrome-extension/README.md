# Aliexpress RU to COM/US switcher

- This **Open-Source** extension fixes the common problem (which Aliexpress has been ignoring for years), so users from `Armenia, Azerbaijan, Belarus, Georgia, Kazakhstan, Kyrgyzstan, Moldova, Russia, Tajikistan, Turkmenistan, Uzbekistan, Ukraine, etc` are not forced into `.ru` domain and can switch to the global version (.com | .us) version.

- It also sets the shipment country for delivery and currency. 

![Screenshoot](https://raw.githubusercontent.com/Puvox/aliexpress-ru-switcher-com-us/refs/heads/main/chrome-extension/screenshot-640.png)

## Install

- For **Chromium browsers**:  Download from Google Store or [Github](https://github.com/Puvox/aliexpress-ru-switcher-com-us) (if you use github, unpack the folder, so you can use load the folder from `Extensions` settings when `Developer Mode` is turned on). 
- For **Tampermonkey**: Download from [GreasyFork](https://greasyfork.org/en/scripts/569775-aliexpress-global-switcher) or manually use the script inside `tampermonkey` folder.

## Usage

Visit aliexpress website, click Extension button in Browser toolbar and choose desired country.
